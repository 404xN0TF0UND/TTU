# TTU Notes - Quick Start Guide

## 🚀 **Get Started in 5 Minutes**

### **Option 1: Portable Package (Easiest)**
1. **Download** `TTU_Notes_Portable.zip`
2. **Extract** to any folder
3. **Double-click** `Start_TTU_Notes.bat`
4. **Open browser** to `http://127.0.0.1:5000`

### **Option 2: From Source Code**
```bash
# Clone repository
git clone https://github.com/yourusername/TTU.git
cd TTU

# Run setup
python setup.py

# Start application
python app.py
```

### **Option 3: Docker**
```bash
# Pull and run
docker run -p 5000:5000 your-registry/ttu-notes
```

## 🎯 **First Steps**

1. **Add Devices** - Go to Devices → Add New Device
2. **Create Templates** - Use Smart Template Suggestions
3. **Execute Commands** - Run commands on your devices
4. **Take Notes** - Document your work

## ⌨️ **Keyboard Shortcuts**

- `Ctrl+/` - Show all shortcuts
- `Ctrl+H` - Go to Home
- `Ctrl+N` - Go to Notes
- `Ctrl+Q` - Quick Notes
